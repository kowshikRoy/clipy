# Contributing to Clipy

Thank you for your interest in contributing to Clipy! We welcome contributions from everyone.

## How to Contribute

### Reporting Bugs
If you find a bug, please create a new issue on GitHub describing the problem, steps to reproduce it, and the expected behavior.

### Suggesting Enhancements
Have an idea? Open a GitHub issue to discuss it!

### Pull Requests
1. **Fork** the repository on GitHub.
2. **Clone** your fork locally.
   ```bash
   git clone https://github.com/YOUR_USERNAME/clipy.git
   ```
3. Create a **new branch** for your feature or fix.
   ```bash
   git checkout -b feature/amazing-feature
   ```
4. **Commit** your changes.
   ```bash
   git commit -m "Add some amazing feature"
   ```
5. **Push** to your fork.
   ```bash
   git push origin feature/amazing-feature
   ```
6. Open a **Pull Request** on the main repository.

## Development

- Open `Clipy.xcodeproj` in Xcode.
- Ensure you have the latest version of Xcode installed.
- Format your code to match the existing style.

### Running Tests

We have automated E2E tests to verify the core features. Please make sure to run them before submitting a PR.

1. Open `Clipy.xcodeproj` in Xcode.
2. Select the `Clipy` scheme.
3. Press `Cmd+U` or go to `Product > Test`.
